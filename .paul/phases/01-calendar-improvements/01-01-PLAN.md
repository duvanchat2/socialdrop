When displayed on the calendar
Then Instagram=pink(#E1306C), TikTok=black(#000000), Facebook=blue(#1877F2), Twitter=blue(#1DA1F2), YouTube=red(#FF0000)

Given a day cell with more than 3 posts
When rendered in month view
Then max 3 events show, and a "+N más" pill appears for the rest (N = count of hidden posts)

## AC-3: Hover tooltip
Given a calendar event
When the user hovers over it
Then a tooltip appears showing: platform name, time (es-CO locale), and truncated caption (max 80 chars)

## AC-4: Persisted view toggle
Given the user selects week or month view
When the page is refreshed
Then the same view is restored from localStorage key 'calendar-view'
